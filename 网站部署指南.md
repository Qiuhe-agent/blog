## 服务器配置

### Nginx配置
配置文件位置：`/etc/nginx/sites-available/qiuheagent.top`

主要配置内容：
- 监听端口：80（HTTP）和 443（HTTPS）
- 网站根目录：/var/www/qiuheagent.top
- 默认首页：index.html
- Gzip压缩：已启用
- 安全头：已配置
- 静态资源缓存：已配置

### SSL证书
- 证书类型：Let's Encrypt免费证书
- 证书位置：/etc/letsencrypt/live/qiuheagent.top/
- 自动续期：已配置（每90天自动续期）

## 故障排查

### 网站无法访问

1. **检查服务器状态**
   ```bash
   ping 115.190.238.160
   ```

2. **检查Nginx服务**
   ```bash
   systemctl status nginx
   ```

3. **检查端口监听**
   ```bash
   netstat -tuln | grep -E '80|443'
   ```

4. **检查配置文件**
   ```bash
   nginx -t
   ```

### 证书问题

1. **检查证书状态**
   ```bash
   certbot certificates
   ```

2. **手动续期证书**
   ```bash
   certbot renew
   ```

3. **重新安装证书**
   ```bash
   certbot --nginx -d qiuheagent.top
   ```

### 域名解析问题

1. **检查域名解析**
   ```bash
   nslookup qiuheagent.top
   ```

2. **清除本地DNS缓存**
   ```bash
   # macOS
   sudo dscacheutil -flushcache
   ```
